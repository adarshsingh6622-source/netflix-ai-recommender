# 🎬 Netflix AI Recommendation System

A production-style Netflix movie recommendation system built using **TensorFlow, FAISS, and Streamlit**.

---

## 🚀 Features

- 🔍 Search movies by name
- 🤖 AI-based recommendations
- ⚡ Fast similarity search using FAISS
- 🎬 Streamlit interactive UI

---

## 🧠 Tech Stack

- Python
- TensorFlow
- FAISS
- Pandas / NumPy
- Streamlit
